## 📌 Sobre o Repositório

Este espaço é dedicado a documentar meus estudos e experimentações na construção e estruturação de APIs utilizando Python. O material aqui presente é fruto das práticas do curso de Análise e Desenvolvimento de Sistemas na UMFG, além de aprofundamentos autônomos na área de back-end.

O fluxo de trabalho documentado neste repositório é centralizado no desenvolvimento local e estruturado:
- **PyCharm:** Utilizado como IDE principal para a escrita dos scripts, gerenciamento do ambiente virtual, testes locais de rotas e versionamento do projeto
